OurCup Recommender
==================

An interactive that allows readers to select a US location (or autodetect it) and show recommendations for World Cup games to watch based on the biggest immigrant populations.

## Developing

This was developed in Visual Studio Code.

1. Make sure you have `npm` installed (`brew install npm` on MacOS).
2. Install the dependencies by running `npm run install`.

## Deploying

1. Build the static JS and CSS to use in the larger story by running `npm run build`.
2. Copy the generated JS and CSS files in `dist` to the right places in the main story.
